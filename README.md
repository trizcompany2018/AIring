# 📝 TRIZ Live Script Generator 🚀

> **제품 정보를 바탕으로 전문적인 라이브 방송 큐시트 및 요약본을 생성하는 AI 어시스턴트**
>
> 본 서비스는 트리즈컴퍼니의 축적된 데이터와 방송 노하우를 기반으로 최적화된 대본을 제공합니다.

---

## ✨ 주요 기능 (Key Features)

* **📦 제품 정보 요약**: 카탈로그(PDF, JPG, PNG)를 업로드하면 제품의 핵심 스펙과 강점을 AI가 즉시 요약합니다.
* **🎬 방송 큐시트 생성**: 트리즈의 성공적인 이전 방송 데이터를 학습한 AI가 전문적인 **60분 방송 큐시트**를 제작합니다.
* **🖼️ 멀티 포맷 지원**: 문서뿐만 아니라 상세페이지 이미지도 분석하여 텍스트를 추출합니다.
* **📄 PDF 다운로드**: 생성된 대본을 깔끔한 레이아웃의 PDF 파일로 즉시 소장할 수 있습니다.

---

## 🛠 기술 스택 (Tech Stack)

### Frontend
- **React**, **Styled-components**, **Axios**
- **React-Markdown** (대본 렌더링)
- **jsPDF**, **html2canvas** (PDF 생성)

### Backend
- **Node.js (Express)**
- **Anthropic Claude 4** (AI Engine)
- **Multer**, **pdf-parse** (파일 처리)

---

## 🚀 작동 방식 (How It Works)

1.  **Upload**: 분석할 제품의 상세페이지나 카탈로그 파일을 업로드합니다.
2.  **Analysis**: Claude AI가 트리즈의 **방송 표준 가이드라인**에 맞춰 데이터를 분석합니다.
3.  **Result**: 생성된 요약본 또는 큐시트 결과물을 확인합니다.
4.  **Download**: 내용을 복사하거나 PDF로 다운로드하여 방송 준비에 활용합니다.
